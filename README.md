# Stone Mill – UP Quarry Search

Interactive dashboard for tracking Upper Peninsula quarry prospects from the Michigan Dimension Stone Feasibility Study (EGLE GIMDL-GGDS00, 1999).

## Features
- Interactive map (OpenStreetMap) with color-coded pins
- Filter by suitability, county, stone type, or top-11 scored sites
- Click any site to see full details with tappable phone/GPS links
- Sortable table

## Colors
- 🟢 Green = VIABLE
- 🟠 Orange = VERIFY
- 🔴 Red = NOT VIABLE

## To use
Open `index.html` in Chrome, or host via GitHub Pages.

## GitHub Pages setup
1. Push this folder to a GitHub repo
2. Go to Settings → Pages → Source: main branch / root
3. Your app will be live at `https://yourusername.github.io/repo-name`
